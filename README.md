## Program 8266 board

```
* Connect the borad TX->RX, RX->TX, VCC->VCC, GND->GND
* Press both (RESET & FLASH/GPIO0) buttons at the sametime
* Release RESET first and then FLASH/GPIO0 button
* Upload the program
* When you release the RESET button, the blue LED will be lit dimly, this means it's ready to bootload
* for moere details https://arduino-esp8266.readthedocs.io/
```
